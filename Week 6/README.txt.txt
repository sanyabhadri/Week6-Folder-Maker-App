# Week 6 Folder Maker App

## Purpose
Explain what your app creates and why it is useful.
This app creates a project folder and the files needed for a simple app idea. It is useful because it helps organize everything in one place instead of making the folders and files by hand.

## How To Run
Explain how to open and run the VB.NET project.
1. Open the GitHub repository.
2. Download or clone the repository.
3. Open the VB.NET solution in Visual Studio.
3. Press Start (or F5) to run the application.
4. Enter the project name.
5. Click the button to create the project structure.
6. The application will automatically generate the required folders and files.

## Folder Structure Created
Docs – Stores Refs, Word and PDF.
Assets – Stores Art and Images.
README.txt – Contains information about the project and how it is organized.

## Story to App Connection
Explain how your story or backstory led to this utility:
The BeatBridge app is a music discovery application designed to help users find new artists and organize playlists. Before building the application, it is important to keep project files organized. This folder maker utility creates a clean project structure so all design documents, source code, screenshots, and assets are stored in the correct locations from the beginning of development.
